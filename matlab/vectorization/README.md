# 数某矩阵中符合某条件的元素数目
例如，想找出矩阵```m```中大于8的元素的数目，可以用

    sum(m(:) > 8)

# 计算行和或列和
经实验，用1向量左乘或右乘矩阵比```sum```函数快。如果是行和，可快5%，如果计算列和，会快30%。用```t_sum_row```来表示用```sum```函数求行和，其他变量定义类似，则有如下关系

    t_sum_col > t_sum_row > t_1_row > t_1_col

MATLAB矩阵是按列存储的，```t_sum_col > t_sum_row```十分奇怪，但事实如此。
